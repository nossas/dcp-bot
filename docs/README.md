# Documentação do projeto

Esta pasta contem a documentação tecnica do chatbot "Defesa Climatica Popular".
Os arquivos estao organizados por tópico e usam links relativos para navegação.

## Como navegar

- Comece por `visão-geral.md` para entender o que o bot faz e os fluxos principais.
- Em seguida leia `arquitetura.md` para o mapa de componentes e integrações.
- Use `configuração-rasa.md` para entender intents, slots, pipeline e policies.
- Consulte `ações-customizadas.md` para ver a lista detalhada de actions.
- Veja `conector-whatsapp.md` para o canal customizado de entrada e saida.
- Use `dados-treinamento.md` para orientar manutenção do treino.
- Veja `banco-de-dados.md` para o esquema local e tracker store.
- Para operação e deploy, use `deploy.md` e `operação-e-manutenção.md`.
