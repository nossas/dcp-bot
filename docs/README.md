# Documentação do projeto

Esta pasta contém a documentação técnica do chatbot "Defesa Climática Popular".
Os arquivos estão organizados por tópico e usam links relativos para navegação.

## Como navegar

- Comece por `visao-geral.md` para entender o que o bot faz e os fluxos principais.
- Em seguida leia `arquitetura.md` para o mapa de componentes e integrações.
- Use `configuracao-rasa.md` para entender intents, slots, pipeline e policies.
- Consulte `acoes-customizadas.md` para ver a lista detalhada de actions.
- Veja `conector-whatsapp.md` para o canal customizado de entrada e saída.
- Use `dados-treinamento.md` para orientar manutenção do treino.
- Veja `banco-de-dados.md` para o esquema local e tracker store.
- Para operação e deploy, use `deploy.md` e `operacao-e-manutencao.md`.
